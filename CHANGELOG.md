# Changelog

All notable changes to FialhoClean will be documented in this file.

## [1.0.0] - 2026-03-23

### Added

- System cleaner with 16 scan categories (temp files, browser cache, prefetch, logs, etc.)
- Windows debloater for removing pre-installed UWP apps
- System optimizer (startup management, services, power plans, privacy, scheduled tasks, GPU HAGS, RAM)
- Repair tools (SFC, DISM, system restore points)
- Dashboard with system info, disk usage, and cleaning history
- Settings with language selection (PT-BR / EN), default categories, and cleaning confirmation
- Dark theme with Carbon Black + Pure White palette
- Custom title bar with minimize/maximize/close
- In-app bug report and GitHub links
